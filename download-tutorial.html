Download Site Tutorial

This page will aquaint you with the basic information needed to operate your Download Dashboard.

Click each section to read the tutorial.

How to Sign In

  • Go to kamyarphotodownload.com and you will arrive at the login page.

  • On first login you will have to enter a new password.

  • Your password must contain:
    • A Lowercase Letter
    • An Uppercase Letter
    • A Number
    • A Symbol

  • We recommend easy to remember phrases such as:

  • brucewayneisnt100%BAT

How to Sign Out

  • To sign out, hit the 'Sign Out' Button at the top right of the page.

  • You will be redirected to kamyarphoto.com

How to Use the Top Navigation

  • The 'Main Website' link will take you to kamyarphoto.com

  • The 'Data Viewer link will take you to the login for our High Accuracy Data Viewer.
    • This platform allows you to edit and view your site in 2D or 3D.
    • You can take measurements of your site and keep track of site progress.

How to View Your User Information

  • Near the top of the page, click 'Click To Show User Information'

  • This will expand a box that contains the user information we have on file:
    • Name
    • Email
    • Phone Number
    • Your Current Project's Unique Identifier

Sorting Files

  • You can sort the table by:
    • File Name
    • Flight Date
    • Upload Date

  • To sort, hit the arrows at the top of each column. The arrow will switch directions each time it's clicked. This switches the table between ascending or descending order.

Pages

  • By default, the table will show you your first 10 most recent files.

  • If you have more results than 10, you can go further into the list by selecting the next page of files at the bottom right of the table.

  • You can also increase the number of files per page using the dropdown at the top left of the table.

Missing a file?

  • If you're expecting a file in the list and it's not there, you can click the 'Missing a file?' button to submit a form for us to find your file.

  • Please keep in mind that it takes 24 to 48 hours to finish editing the data after the flight takes place.

Download Files

  • To download a file, simply click on the file name you would like to download.

  • All files are uploaded in '.zip' format.

  • The file size is listed at the right most column inthe table, this will give you an idea on how long it will take to download based on your experience with your internet speed.

  • You will need to extract it and load it with a program compatible with the data file type.

File Information

  • The information shown for each file includes:
    • File Name
    • Flight Date
    • Upload Date
    • File Size
  • We always recommend using the latest browser, that way we can ensure you have a consistent experience.

  • If you must use an older version of a browser, our sharing platforms are only compatible with the following:

    • Data Viewer
      • Chrome Desktop Version 64 or Higher

    • Download Site
      • Chrome Desktop Version 64 or Higher
      • Edge Desktop Version 79 or Higher
      • Firefox Desktop Version 78 or Higher
      • Safari Desktop Version 12 or Higher

  • If you experience Technical Issues on the website, please email service@kamyarphoto.com with a description of what happened. We will take care of you as soon as possible.
